# Cheon Smart Planner

## Setup

### Android

#### Production build

`flutter run --release --flavor prod`

#### Development build

`flutter run --release -t lib/main_dev.dart --flavor dev`

### IOS

#### Production build

`flutter run --release`

#### Development build

`flutter run --release -t lib/main_dev.dart`

### Static analysis

`flutter analyze`

### Automated tests

`flutter test`
